# 產品需求文件 (PRD)

**專案名稱**：CS2 賽事數據與飾品投資分析系統
**專案描述**：結合職業電競賽事戰績追蹤，以及虛擬飾品（如刀、槍枝外觀）市場價格波動分析的綜合情報平台。

## 1. 專案概述
**背景與動機**
近期《絕對武力 2》(CS2) 的電競賽事持續火熱，且遊戲內虛擬飾品（如刀、手套、槍枝外觀）擁有龐大的市場與真實的交易價值。然而，目前玩家與交易者往往需要在多個不同平台之間切換，才能同時獲得最新的賽事戰績與飾品的市場價格走勢。為解決資訊破碎的問題，我們希望建立一個綜合情報平台，整合賽事戰績追蹤與飾品投資分析，幫助使用者做出更好的觀賽決策與投資規劃。

**目標用戶**
- FPS 遊戲玩家
- 電競賽事觀眾
- 虛擬飾品交易/投資者

**核心價值主張**
在單一平台即時掌握最新 CS2 職業賽事動態，同時能追蹤虛擬飾品市場的價格波動，並管理個人投資組合，為玩家與投資者提供專業、客觀且全面的綜合情報服務。

## 2. 功能需求

以下列出本系統的主要功能與對應的使用者故事 (User Story)：

1. **職業戰隊與選手戰績查詢**
   - 作為 **電競賽事觀眾**，我希望 **能快速查詢職業戰隊與選手的近期賽事戰績**，以便 **掌握隊伍近況、選手發揮狀況及賽事走勢**。

2. **虛擬飾品市場價格圖表與排行**
   - 作為 **虛擬飾品交易者**，我希望 **能觀看各項虛擬飾品市場價格的歷史走勢圖表與漲跌幅排行**，以便 **分析市場行情趨勢並尋找潛在的投資標的**。

3. **個人投資組合追蹤**
   - 作為 **虛擬飾品交易者**，我希望 **能記錄自己持有飾品的買入價位與數量，並追蹤當前市值**，以便 **隨時掌握個人投資組合的具體損益狀況**。

4. **飾品到價提醒功能**
   - 作為 **虛擬飾品交易者**，我希望 **能在關注的飾品價格達到或跌破我指定的目標價時收到通知提醒**，以便 **能把握最佳的買進與賣出時機，避免錯失交易機會**。

5. **社群討論與賽事預測**
   - 作為 **FPS 遊戲玩家**，我希望 **能有一個社群討論版，並能針對即將到來的賽事參與勝負預測投票**，以便 **與其他玩家交流觀賽心得、討論戰術與分享投資看法**。

## 3. 非功能需求

**技術限制**
- 系統架構限制使用 Web 框架進行開發（例如：Flask + Jinja2）。
- 資料庫採用 SQLite，方便部署與輕量級開發管理。

**效能與安全考量**
- **安全性**：必須對使用者輸入數據進行驗證，防止 SQL Injection 與 XSS 攻擊；使用者密碼必須經過雜湊 (Hash) 處理後儲存。
- **效能**：高頻查詢的資料表（如價格走勢與戰績）需建立索引以提升存取速度。圖表渲染應使用前端輕量化的可視化套件。
